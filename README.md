# Bf-rs

### A simple [Brainfuck](https://en.wikipedia.org/wiki/Brainfuck) interpreter in rust

## Quick Start
```shell
$ git clone git@github.com:DuskyElf/bf-rs.git
$ cd bf-rs
$ cargo run --release example.bf
```
